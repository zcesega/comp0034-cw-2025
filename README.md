Please note that code will be downloaded, setup and run using automated scripts using config in
`pyproject.toml` and `submission.yml`. Please refer to the coursework specification document for
details.

Bespoke instructions in this readme.md are not likely to be read by the marker for this coursework
due to the volume of student work to be assessed. Thank you.

Please use the src layout for your project e.g.:

```text
project_root/
├── pyproject.toml
├── README.md
├── submission.yml
├── src/
│   └── example_package_name/
│       ├── __init__.py
│       └── example_app.py
└── tests/
```

Avoid spaces, numerals and non-English characters in file and folder names.
